## INSTALLATION

Add via _composer_

    composer require chriscubos/hello

Add to _config/app.php_ at the providers []:

    Chriscubos\Hello\HelloServiceProvider::class,

## USAGE

```php
    Hello::world();
    Hello::there();
    Hello::friend('John');
    Hello::message('whatever you want to do');
    Hello::target($target, $action);
```
